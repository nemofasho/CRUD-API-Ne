# CRUD-API-Ne
CSC 340 assignment 3, learning APIs


# Installation
- Get the project
- clone
    - git clone // https://github.com/csc340-uncg/f25-jpa-crud-api.git
OR download zip.
Open the project in VS Code.
This project is built to run with jdk 17.
Dependencies to JPA and Postgres in addition to the usual Spring Web. JPA handles the persistence, Postgresql is the database to be used.
/src/main/resources/application.properties This file has the configuration for the PostgreSQL database to use for the API.
You MUST have the database up and running before running the project!
Login to your neon.tech account.
Locate your database project.
On the project dashboard, click on "Connect" and select Java.
Copy the connection string provided.
Paste it as a value for the property spring.datasource.url. No quotation marks.
Build and run the main class. You should see a new table created in the Neon database.

2. /{animalID} (GET)
Gets an individual animal in the system identified by a random id.

# Parameters
Path Variable: studentid <Long> - REQUIRED

# Response - A single animal

{
  "animalId": 3,
  "name": "blocto",
  "age": 4,
  "description": "One of the most dangerous octopus in the world",
  "animalType": "blur ringed octopus"
}

5. / (POST)
Create a new animal entry.

# Request Body
An animal object. Note the object does not include an ID as this is autogenerated.

{    "name": "princess",
    "age": 4,
    "description": "My small pet from home",
    "animalType": "dog"
}

# Response - The newly created animal
{
  "animalId": 11,
  "name": "princess",
  "age": 4,
  "description": "My small pet from home",
  "animalType": "dog"
}

6. /{animalId} (PUT)
Updates an existing animal.

# Parameters
Path Variable: animalId <Long> - REQUIRED

# Request Body
An animal object with the updates.
{
  "animalId": 11,
  "name": "princessupdated",
  "age": 5,
  "description": "My small pet from home",
  "animalType": "dog"
}

# Response - The updated animal object.
{
  "animalId": 11,
  "name": "princessupdated",
  "age": 5,
  "description": "My small pet from home",
  "animalType": "dog"
}
